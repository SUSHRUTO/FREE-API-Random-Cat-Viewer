# 🐱 CatVerse – Professional Random Cat Viewer

CatVerse is a modern and interactive Random Cat Viewer Application built using HTML, Tailwind CSS, and Vanilla JavaScript.

The application fetches random cat images dynamically using the FreeAPI Random Cat endpoint and presents them in a clean, engaging, and professional user interface.

It also includes random cat generation, cat facts, image download functionality, recently viewed gallery, responsive design, and smooth UI interactions.

# ⚙️ Functionalities

## 🐱 Dynamic Random Cat Viewer

- Fetches random cat images dynamically from FreeAPI
- Displays cat images in a professional UI
- Updates image on every button click

---

## 🎲 Random Cat Generator

- Generates new random cat images dynamically
- Displays unique cat titles and descriptions
- Prevents image cropping using object-contain

---

## 🖼️ Recently Viewed Gallery

- Stores recently viewed cat images
- Displays them in a responsive gallery
- Automatically limits gallery size

---

## 📥 Download Image Functionality

Users can:
- Download displayed cat images
- Save images locally

---

## 🧠 Cat Facts Generator

- Displays random cat facts dynamically
- Updates fact on button click

Example Facts:

```bash
Cats sleep for around 13 to 16 hours a day.
```
